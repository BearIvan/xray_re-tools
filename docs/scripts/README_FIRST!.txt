=== readme для perl скриптов (trunk/scripts) ====

В папке лежат скрипты трех разновидностей: дамперы (папка dumpers), распаковщики (папка unpackers) и 
служебные скрипты (service). Дамперы предназначены для вывода содержимого файлов сталкера в окно консоли, 
распаковщики - для распаковки этих файлов в более понятный формат (обычно ltx-конфиги), а служебные 
скрипты - для облегчения модифицирования игры. Для корректной работы любого perl скрипта 
необходимо установить интепретатор ActivePerl (http://www.activestate.com/activeperl/downloads) 
и поместить папку stkutils в папку со скриптом.

Состав папки dumpers:
	-level1xxx/: дамп файлов из папки уровня билдов 1098-1154. 
	-level/: дамп файлов из папки уровны финалки и остальных билдов.
	-SDK/: дамп файлов-исходников уровней.
	-misc/: дамп некоторых ресурсных файлов (anm, dm, ogf, ppe).

Состав папки unpackers:
	-game_graph/: скрипты для работы с глобальным графом игры.
		gg_cdc_zp.pl - распаковщик/запаковщик глобального графа ЧН и ЗП.
		ggrc.pl - скрипт для распаковки и совмещения графов и переноса локаций между разными 
версиями игры.
	-spawn/: скрипты для распаковки/запаковки all.spawn разных версий игры.
	-level/: скрипты для работы с некоторыми файлами уровней.
		-level
		-level.game
		-level.env_mod
		-level.ps_statis
		-level.snd_statis
	-xr_formats/: скрипты для работы с некоторыми файлами формата *xr.
		-gamemtl.xr
		-lanims.xr
		-seinvironment.xr
		-shaders_xrlc.xr

Состав папки service:
	-resource_copier/ - копирование рабочей карты из СДК вместе с ресурсами, относящимися к ней.
	-chunk_parser/ - разбор файла RIFF-формата на составные чанки.

Для более подробной информации смотрите соответствующие ридми в этой папке.
Примечание: При использовании утилит из папки xiani, используйте папку stkutils.xiani, переименовав ее
в stkutils.
