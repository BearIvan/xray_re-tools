Скрипт для копирования рабочих ресурсов локации из СДК. (Resource Copier v.0.4)

Копирует:
-папку с заданным уровнем из X:\rawdata\levels
-папку с заданным уровнем из X:\rawdata\objects\levels
-текстуры и бампы (dds, thm, tga), используемые на локации.
-звуки (ogg), используемые на локации.
Скрипт ведет лог, в который записывает копируемые текстуры, звуки и объекты вместе со статусом копирования
(скопировано или нет, если нет, то почему). Лог находится в X:\logs

Использование: 
	perl rc.pl -c <level_name> -o <destination_dir> -tga -soc

Ключ -tga предназначен для копирования текстур tga, которое сделано опциональным.
Ключ -soc использовать для копирования ресурсов из СДК ТЧ.
В файл ignore.ini можно прописывать игнорируемые объекты, звуки и текстуры, формат такой:

[objects]
editor\artefakt_ah

[game_textures]
act\act_corp_military

[game_sounds]
ambient\air_2

Важные замечания:
-для сдк должен быть смонтирован диск X.
-в папке X:\logs должен быть лог от Level Editor, причем только текущего пользователя. 
Если в папке лежат логи разных пользователей LE, скрипт будет читать имена текстур из всех логов.
-при указании в ignore.ini имени текстуры, соответствующие тга и тхм также не будут копироваться.
